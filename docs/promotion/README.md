## Pitch (short)

GitPulse is an open-source **tray/menu bar app** for GitHub: grouped issues and PRs, stars (yours + your repos by popularity), watching, notifications, activity feed, background polling with desktop alerts. **Tauri 2 + Vue 3 + Rust**, MIT, macOS / Windows / Linux.

**Ask:** What would make this your daily driver?
