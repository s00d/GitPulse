## Text (optional comment / first reply)

```
I built GitPulse because I kept opening GitHub just to check assigned issues and PR review queues.

GitPulse is a cross-platform tray app (macOS/Windows/Linux) built with Tauri 2 + Vue 3. It groups issues and PRs by repo, shows starred repos and your own repos by star count from the tray, polls in the background, and sends desktop notifications when something changes.

Sign-in via GitHub device code, gh CLI, or PAT. MIT licensed.

I'd love feedback on what would make this a daily driver — especially compared to GitHub Mobile or email notifications.

Demo gif: https://github.com/s00d/GitPulse#gitpulse
Releases: https://github.com/s00d/GitPulse/releases/latest
Feedback thread: https://github.com/s00d/GitPulse/discussions/1
```
