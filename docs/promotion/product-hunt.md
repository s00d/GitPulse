## Description

```
GitPulse keeps your GitHub work one click away in the system tray.

Instead of opening github.com all day, see grouped issue and PR counts by repository, starred repos, watching, and unread notifications from the menu bar. Open a focused dashboard when you need detail — Overview, Feed, Issues, PRs, Stars, Watching, Notifications.

Built for developers who live in GitHub:
• Tray-first workflow — close the window, stay in the tray
• PR queues: needs review, my PRs, waiting on author
• Background refresh with desktop notifications on changes
• Hide repos from the app and tray per-project
• Sign in with device code, GitHub CLI, or token

Open source (MIT), cross-platform: macOS, Windows, Linux.
Built with Tauri 2 + Vue 3 + Rust.
```
